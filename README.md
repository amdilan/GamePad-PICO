#### Gamepad



###### Microcontroller : RP2040

###### Programming Language : CircuitPython 8

###### Other Modules:

* ###### PCF8574
* ###### ADS1115



This is a **RP2040** based gamepad; which uses Adafruit **CircuitPython** 8.2.10 and adafruit Libraries.

Used **PCF8574** I/O Expander module to expand the lacking I/O of RP2040, and used **ADS1115** ADC module for reading inputs from dual joysticks.
